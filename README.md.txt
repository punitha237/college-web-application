College web application

Project overview
A College website designed to simplify interactions between students, faculty, parents, and administrators. The application also highlights the college's facilities, infrastructure, and reasons to join the institution.

Features
- College information and overview
- Reasons to join the college
- Facilities and infrastructure section
- Student Dashboard
- Faculty Dashboard
- Admin Dashboard
- Responsive design
- Light/Dark mode support

Technologies Used
- HTML
- CSS
- JavaScript
- Antigravity AI

Installation Steps
1. Download the project files.
2. Open the project folder.
3. Open `index.html` in a browser.
